
# 🚀 Desafio Nasajon

Projeto desenvolvido como solução para o desafio técnico proposto pela **Nasajon**, com o objetivo de avaliar habilidades em desenvolvimento, lógica e organização de código.

---

## 📌 Sobre o projeto

Este repositório contém a implementação do desafio, abordando conceitos importantes como:

* Manipulação de dados
* Consumo/processamento de informações
* Estruturação de código
* Boas práticas de desenvolvimento

O projeto foi desenvolvido com foco em clareza, organização e eficiência.

---

## 🛠️ Tecnologias utilizadas

* TS / Node.js 


---

## ⚙️ Funcionalidades

* ✔️ Leitura e tratamento de dados
* ✔️ Processamento das informações conforme o desafio
* ✔️ Estrutura organizada e modular

## ▶️ Como executar o projeto

### 1. Clone o repositório

```bash
git clone https://github.com/VitorPio7/Desafio-Nasajon.git
```

### 2. Acesse a pasta

```bash
cd Desafio-Nasajon
```

### 3. Instale as dependências

```bash
npm install
```

### 4. Execute o projeto

```bash
npm start dev
```
