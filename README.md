## ▶️ Como executar o projeto

### 1. Clone o repositório

```bash
git clone https://github.com/VitorPio7/Desafio-Nasajon.git
```

### 2. Acesse a pasta

```bash
cd Desafio-Nasajon
```

### 3. Instale as dependências

```bash
npm install
```

### 4. Execute o projeto

```bash
npm start
```
